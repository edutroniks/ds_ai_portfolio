    About Me

    Hi, I’m Marek, an early‑stage Data Science and Machine Learning practitioner focused on building clear, reliable, and practical solutions. My work combines structured analysis, reproducible workflows, and user‑oriented design to create tools that genuinely help people make better decisions.

    I enjoy turning complex problems into understandable, actionable insights. Whether I’m exploring a dataset, designing a model, or building an interactive app, I aim for clarity, transparency, and long‑term maintainability.

    How I Work¶

    I design reproducible ML workflows with clean separation between data preparation, model training, and deployment.

    I focus on clear insights, making sure that results are understandable and meaningful for real users.

    I build practical tools, from exploratory analyses to full applications, always with usability in mind.

    I value collaboration, structured communication, and thoughtful documentation.

    I work collaboratively with AI tools such as GitHub Copilot. I don’t create every line of code myself, but I understand, review, and correct the generated code, and I use it to build clear, reproducible, and practical solutions.

    What I’m Interested In¶

    • Machine learning that solves real problems

    • Interactive applications that make data accessible

    • Clean, maintainable code and transparent workflows

    • AI agents and modern automation tools

    • Visual clarity and user‑centered design

    Beyond the Code¶

    I enjoy learning, experimenting, and refining my workflow. I treat each project as an opportunity to improve both the technical and the human side of data science: communication, structure, and design.

    If you’d like to collaborate or discuss an idea, feel free to reach out through GitHub.
